## ✨ What is 2D Animation Studio?

**2D Animation Studio** is a free, open-source animation tool that runs entirely in your browser. Whether you're a YouTuber, educator, content creator, or just someone who loves animation — 2D Animation Studio gives you the power to create expressive character animations without spending a single rupee.

Built with pure HTML, CSS, and JavaScript. No frameworks. No servers. No subscriptions.

## 🕹️ How to Use

1. **Choose a character** from the left panel
2. **Customize** skin tone, hair color & outfit
3. **Move the sliders** on the right panel to pose the character
4. **Press `K`** or click "Add Keyframe" to save the pose
5. **Move to a new frame** on the timeline
6. **Pose again** and add another keyframe
7. **Press `Space`** to play the animation
8. **Click Export** to download your animation as a video

---

## ⌨️ Keyboard Shortcuts

| Key | Action |
|---|---|
| `Space` | Play / Pause |
| `K` | Add Keyframe |
| `← →` | Move frame by frame |
| `Home` | Go to first frame |
| `End` | Go to last frame |

- **CSS3** — Dark UI with smooth animations
- **MediaRecorder API** — Video export
